# Movie Recommendation System

## Introduction
This project is a Movie Recommendation System developed using Machine Learning. The system recommends movies based on similarity between movie genres using cosine similarity.

## Features
- Recommend movies based on genre
- Uses Machine Learning algorithm
- Content-based recommendation system
- आसान और simple user input system

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NumPy

## Dataset
The dataset used in this project is movies.csv which contains movieId, title, and genres.

## How to Run the Project
1. Open terminal in project folder
2. Run the command:
   python main.py
3. Enter movie name
4. The system will recommend similar movies

## Algorithm Used
Cosine Similarity is used to measure similarity between movies.
